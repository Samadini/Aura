# Aura - AR Beauty Try-On App

## Description
Aura is an innovative iOS application that leverages AR technology to provide a virtual makeup try-on experience. The app allows users to visualize how different makeup products, particularly lipsticks, would look on their face in real-time using the device's front camera.

## Features
- **AR-Powered Virtual Try-On**: Real-time lipstick application using ARKit face tracking
- **Shade Matching**: Smart algorithm to suggest perfect lipstick shades
- **Product Catalog**: Browse and try on various lipstick collections
- **Lighting Adjustment**: Adaptive rendering for different lighting conditions
- **Save & Share**: Save favorite looks and share with friends
- **User Profiles**: Personalized recommendations based on skin tone and preferences

## Requirements
- iOS 16.0+
- Xcode 14.0+
- Device with TrueDepth camera (iPhone X or newer)
- Swift 5.7+
